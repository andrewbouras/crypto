Create Webhook
Programatically create a Helius webhook.

POST /webhooks
Creates a webhook with specified account addresses, transaction types, and a webhook URL. Optionally, you can include an authorization header to verify that the webhook originated from Helius.  For a full list of supported transaction types, please see Transaction Types

You can add up to 100,000 addresses via the API.

Creates a webhook.
POSThttps://api.helius.xyz/v0/webhooks
Query parameters

Body

application/json
webhookURLstring
transactionTypesarray of TransactionType (enum)

accountAddressesarray of string

webhookTypestring
authHeaderstring
Response

200
The created webhook.

Body

application/json
webhookIDstring
walletstring
webhookURLstring
transactionTypesarray of TransactionType (enum)

accountAddressesarray of string

webhookTypestring
authHeaderstring
Request

    method: 'POST',

      "Content-Type": "application/json"

});
const data = await response.json();

Get All Webhooks
Programatically retrieve all your Helius webhooks.

Due to response size limitations, the number of returned addresses per configuration is capped at 100. To access the complete list of addresses, use our Get Webhook by ID endpoint.

GET /webhooks 
Returns all webhooks.
GEThttps://api.helius.xyz/v0/webhooks
Query parameters

Response

200
Webhooks for the given API key.

Body

application/json
webhookIDstring
walletstring
webhookURLstring
transactionTypesarray of TransactionType (enum)

accountAddressesarray of string

webhookTypestring
authHeaderstring
Request

JavaScript
Copy
const response = await fetch('https://api.helius.xyz/v0/webhooks?api-key=text', {
    method: 'GET',
    headers: {},
});
const data = await response.json();
